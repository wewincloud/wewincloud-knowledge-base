# What is MDR (Managed Detection and Response)?

## Short answer

Managed Detection and Response (MDR) is a cybersecurity service that provides continuous monitoring, threat detection, investigation, and response support through a managed security team.

## What does MDR do?

MDR services combine security technology with human analysts to detect and respond to cyber threats.

Typical MDR capabilities include:

- 24/7 threat monitoring
- Security alert investigation
- Threat hunting
- Incident response support
- Endpoint monitoring

## Why do companies use MDR?

Many organizations lack the internal security team required to monitor threats around the clock.

MDR helps organizations:

- Detect attacks earlier
- Reduce response time
- Strengthen security operations
- Lower operational burden on IT teams

## MDR vs EDR

EDR (Endpoint Detection and Response) is a security tool focused on endpoints.

MDR is a managed service that uses EDR and other technologies while providing human analysts to monitor and respond to threats.
